Q.1  What is Node.js? How does it differ from browser JavaScript?
->  Node.js is a JavaScript runtime environment that lets you run JavaScript outside the browser, mainly on the server side.
It is built on Google Chromeâ€™s V8 JavaScript engine and is commonly used to build backend services, APIs, real-time apps, and CLI tools.
    Difference Between Node.js & Browser JavaScript:

| Aspect              | Node.js                              | Browser JavaScript                |
| ------------------- | ------------------------------------ | --------------------------------- |
| **Environment**     | Runs on the server                   | Runs inside the browser           |
| **Purpose**         | Backend logic, APIs, file handling   | UI interactions, DOM manipulation |
| **APIs Available**  | File system, OS, networking, process | DOM, BOM (window, document)       |
| **Access to Files** |  Direct file system access           |   No direct file access           |
| **Global Object**   | `global`                             | `window`                          |
| **Modules**         | CommonJS / ES Modules                | ES Modules                        |
| **Security**        | Trusted server environment           | Restricted for user safety        |
